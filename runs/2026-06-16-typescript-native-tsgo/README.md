## Reproduce (on your own repo)

```bash
./run.sh /path/to/your/typescript/repo    # defaults to the current directory
```

Needs Node + `npx`. It clears the incremental cache for the cold pass, times both compilers
cold+warm, then prints a speed table **and** a `tsc`-vs-`tsgo` diagnostic diff. Your seconds
will differ from ours (different codebase) â€” that is expected and honest. The signal is the
**ratio** and whether **your** diagnostics diverge.
